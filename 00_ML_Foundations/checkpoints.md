# Module 0 — Concept Checkpoints

Learners should be able to:
1. Explain the ML pipeline end-to-end
2. Differentiate loss vs evaluation metrics
3. Explain why training ≠ inference
4. Identify overfitting from a plot
5. Explain why unsupervised learning has no “correct” answer

Answer these **without looking back** at the notes.

---

## Section A — Core Understanding

1. What is the difference between:
   - a **loss function**
   - an **evaluation metric**

2. Why does a model not “understand” the data it learns from?

3. In your own words, explain the ML pipeline end-to-end.

---

## Section B — Spot the Confusion

4. A model has 98% accuracy on training data and 55% on test data.
   - What is happening?
   - Where in the pipeline does the issue originate?

5. Can an unsupervised model be wrong?
   - If yes, how?
   - If no, why not?

---

## Section C — Explain Like I’m 12

6. Explain Machine Learning to a 12-year-old in **two sentences**.

7. Explain the difference between training and inference using a real-world analogy.
