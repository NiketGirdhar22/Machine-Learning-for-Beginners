# Module 0: ML Foundations

## Purpose of This Module

Most beginners fail at Machine Learning not because algorithms are hard, but because they never build the correct mental model.

This module exists to fix that.

By the end of Module 0, you should:
- Understand what Machine Learning actually is
- Know the end-to-end ML pipeline
- Clearly distinguish training vs inference
- Understand why ML models fail
- Be mentally prepared to learn algorithms in the next modules

***Important:***
Do not skip this module. Every algorithm later depends on these ideas.
